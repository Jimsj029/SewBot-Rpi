# Python 3.x required
# 
# RASPBERRY PI USERS: Do NOT use pip install -r requirements.txt
# Instead, run: ./setup.sh
#
# The setup script installs system packages (python3-opencv, python3-numpy, etc.)
# which are pre-compiled for ARM architecture. This avoids "Illegal instruction" errors.
#
# System packages installed by setup.sh:
#   - python3-numpy (ARM-compatible)
#   - python3-opencv (ARM-compatible)
#   - python3-pygame (ARM-compatible)
#   - python3-matplotlib, python3-pil, python3-yaml, python3-requests, etc.
#
# Pip packages (from piwheels):
#   - torch
#   - torchvision
#   - ultralytics
#   - onnx
#   - onnxruntime
